**qrep_client_qa** — клиентский бот и webhook-сервер (QA)

**Назначение:** упрощённая клиентская версия бота для тестов + сервер для приёма webhook’ов о покупках.

- **Стек:** Python, aiogram, aiohttp, PostgreSQL (async), SQLAlchemy async
- **Состав:**
  - **Бот** — только клиентский функционал (без админов и staff), локализация, задачи по расписанию
  - **Webhook-сервер** (`webhook_server.py`) — приём событий о покупках (`PurchaseWebhookHandler`), логи в `webhook_server.log`
  - **Тесты** — `test_production.py`, `test_notifications.py`
- **Запуск:**
  - Бот: `python bot.py`
  - Webhook-сервер: `python webhook_server.py`
